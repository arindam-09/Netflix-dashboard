1. Title

Netflix Content Analysis Dashboard

2. Short Description / Purpose

This project presents an interactive Power BI dashboard to analyze Netflix’s content library. It helps in understanding trends in genres, release patterns, ratings, and global distribution. The goal is to derive meaningful insights to support content strategy and decision-making.

3. Used Skills

a. Power BI (Dashboarding, DAX, Data Modeling)
b. Data Cleaning and Transformation (Power Query)
c. Data Visualization and Storytelling
d. Exploratory Data Analysis (EDA)
e. KPI Design and Insight Extraction

4. Data Source

a. Public Netflix dataset sourced from platforms like Kaggle
b. Dataset includes fields such as title, genre, release year, country, rating, duration, and content type (Movie or TV Show)

5. Features and Highlights
a. Problem

i. Difficulty in identifying dominant genres across the platform
ii. Lack of clarity on content growth over time
iii. Limited visibility into country-wise content contribution

b. Key Goal

i. Build an interactive and user-friendly dashboard
ii. Analyze trends in content production over time
iii. Identify popular genres and categories
iv. Understand global distribution of Netflix content

c. Walkthrough of Key Visuals

i. Content Distribution (Movies vs TV Shows): Donut or pie chart showing overall composition
ii. Release Trends: Line chart representing content growth across years
iii. Genre Analysis: Bar chart highlighting most frequent genres
iv. Country-wise Contribution: Map or bar chart showing content origin
v. Ratings Analysis: Chart showing distribution of content ratings

d. Business Impact and Insights

i. Helps identify content gaps and potential areas for investment
ii. Provides insights into audience preferences through genre trends
iii. Supports regional content strategy using country-level analysis
iv. Assists in understanding target audience segments through ratings distribution
